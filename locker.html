About Locker

Locker is a HTML based webapp to store information after encryption. It supports storing of 3 types of structured information - passwords, notes, and journals.

How to use

Create a database by entering the desired database name, username, and password. The password is used to encrypt your information. Once the database is created you can add passwords, notes and journals.

Once you have added information, export your information. This file will contain all your information in encrypted form.

To again work on the same information or to add more information, first import the file, unlock it using the set password, and add or edit the information.

Is my information safe

This is one of the safest app. No information is stored on any server. When you add any information, it is temporarily stored in your browser. It is lost once you refresh the page of close the browser. To reuse the information you must export it and store it whereever you like.
